```python
def timeit(func):
    def wrapper(args):
        from datetime import datetime as dt
        start = dt.now()
        result = func(args)
        end = dt.now()
        print('f({}) took time {}'.format(args,
                                          (end - start)))
        return result
    return wrapper
```

## Breakdown

+++

```python
@decorator
def f():
    ...
```

```python
f = decorator(f)
```

`decorator` takes a function as argument, hence

```python
def decorator(func):
    ...
```

But `decorator(f)` also _returns_ a function, so

```python
def decorator(func):
    def inner_function(args):
        ...
    return inner_function

```

+++

```python
def decorator(func):
    def new_function(the, args, to, func_):
        return_value = func(the, args, to, func_)
    return new_function


@decorator
def myfunc(args):
    print('Hello, world!')

```

That's equivalent to ...

+++

```python
def decorator(func):
    def new_function(the, args, to, func_):
        return_value = func(the, args, to, func_)
    return new_function

def myfunc(args):
    print('Hello, world!')
myfunc = decorator(myfunc)
```
